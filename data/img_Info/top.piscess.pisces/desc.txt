Pisces is a media player with the following main features: 1: Playback of local video files, supporting playback of video files such as mp4. 2: Playback of m3u8 streaming media, any streaming media address that conforms to the m3u8 file format specification can be played in Pisces.Pisces is designed as an online m3u8 format media player program, specifically intended for streaming m3u8 and mp4 media.M3u8 is a multimedia playlist file format, Pisces is programmed to handle m3u8 and mp4 streams exclusively, ensuring a seamless streaming experience for our users.  M3u8 is a standard format for specifying a media playlist, essentially functioning as a plain text file that acts as an index for multimedia streams.
